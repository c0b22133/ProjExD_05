# 弾幕ゲー
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
弾幕を避けて敵を倒し、レベルアップ！

## クリア条件
なし！ゲームオーバーになるまで！！

## 操作方法
A:左移動
D:右移動
TAB:シールド生成

## ゲームの実装
### 共通基本機能
* 背景画像と操作キャラクターの描画

### 担当追加機能
* 射撃機能：キャラに追従して自動発射する機能(C0B22133/眞篠琉一)
* キャラクターの中心からビームがフレームに応じて発射
* 途切れることなく移動に忠実
